Content Posting API – Entegrasyon

Bu uygulama, TikTok Content Posting API ile günlük hisse analizi görsellerini @uhyborsa hesabına otomatik paylaşır.

Akış (end-to-end)

1. Zamanlayıcı günde 2 kez çalışır (örn. 08:00 ve 12:00).
2. Telegram botundan günün 2 hissesi için analiz görseli + YZ özeti alınır.
3. Görsel doğrulanmış URL’de (bu site veya CDN) yayınlanır.
4. TikTok Content Posting API (POST /v2/post/publish/content/init/) ile PHOTO post oluşturulur; Direct Post ile doğrudan @uhyborsa hesabına yayınlanır.
5. Kullanıcı TikTok uygulamasında paylaşımı görür.

Kullanılan scope

  • video.publish – İçeriğin doğrudan profil sayfasına yayınlanması için kullanılır.
